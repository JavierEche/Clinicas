# Clinic ERP

Estructura base del proyecto adaptada al esquema solicitado para una aplicación clínica tipo ERP.

## Estructura principal

- `app/` con capas `Http`, `Models`, `Services`, `Repositories`, etc.
- `database/` con `migrations`, `seeders` y `factories`.
- `resources/views/` organizado por dominios funcionales.
- `routes/` con archivos `web.php`, `api.php` y `admin.php`.
- `storage/app/historias_clinicas/` para almacenamiento de documentos clínicos.
- `tests/Feature` y `tests/Unit` para pruebas automatizadas.
